# Beginner Guide

Cette page décrit une **première session réaliste** avec le runtime actuel.

## 1) Premier regard: vérifier l’économie
- Confirme tes stocks de départ (`ore`, `stone`, `iron`).
- Vérifie la capacité: au début, le plafond de stockage est bas, donc le surplus est perdu si tu capes.

## 2) Priorité bâtiments (ouverture)
- **Mine** et **Quarry**: sécurisent le rythme de construction.
- **Refinery**: alimente la ressource avancée (`iron`) utile aux paliers.
- **Warehouse**: évite le gaspillage en augmentant les caps.

## 3) Gérer les files
- Construction: file limitée (2 slots configurés).
- Training: file dédiée unités.
- Research: file dédiée recherches.

Objectif early game: garder la construction active en continu, sans bloquer la population ni saturer le stockage.

## 4) Débloquer la progression militaire
- Monter `barracks` pour les unités terrestres.
- Préparer `research_lab` pour accélérations (production/training/defense).
- Ne pas lancer des unités sans vérifier les prérequis recherche.

## 5) Checkpoints utiles
- Si tes ressources stagnent: prioriser prod + warehouse.
- Si tes files sont vides: préparer coûts en amont puis relancer queue.
- Si la pop est limitante: housing complex avant train massif.
