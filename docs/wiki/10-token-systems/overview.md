# Token systems

**Status:** Design in progress

## What this page covers
Vue d’ensemble des mécaniques tokenisées de Coinage côté joueur : factions token, serveurs token, boosts potentiels et garde-fous économiques.

## How it works
- Les tokens servent d’ancrage communautaire et économique selon les règles de saison.
- Les bénéfices exacts doivent rester lisibles côté joueur et plafonnés pour éviter le pay-to-win.
- Les variantes de serveurs basées sur token modifient la métagame (composition de factions, diplomatie, économie), sans changer les fondamentaux RTS/MMO.
