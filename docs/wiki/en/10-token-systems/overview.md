# Token systems

**Status:** Design in progress

## What this page covers
High-level player-facing overview of Coinage tokenized systems: token factions, token servers, potential holder boosts, and economic safeguards.

## How it works
- Tokens act as a community and economy layer across seasonal worlds.
- Benefits must stay transparent and capped to avoid pay-to-win outcomes.
- Token server variants influence meta choices (faction mix, diplomacy, market pressure) without changing core city-building and warfare fundamentals.

## Requirements
- Show active token-related effects and limits in-game.
- Document differences between standard and token server rulesets.
- Explain impact on alliances, wars, and market systems.
