# Coinage

Browser strategy game foundation built with Next.js (App Router) + TypeScript + PixiJS.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000`.

## Current scope

- Landing page at `/`
- Play route at `/game`
- PixiJS Map View with procedurally generated world (500+ neutral factions)
